✅ To-Do App (Vanilla JS + Express)
A simple, lightweight To-Do web application built using HTML, CSS, and JavaScript (DOM) for the frontend, and Node.js with Express for serving static files on the backend.

✨ Features include adding, deleting, editing, and checking tasks — with in-memory storage (no database).

📸 Preview

🛠️ Features
✅ Add new tasks

❌ Delete tasks

✏️ Edit/save tasks inline

☑️ Mark tasks as complete (strike-through & greyed out)

💾 In-memory task storage (no database)

🧾 Express backend serves all files locally

🔧 Tech Stack
Frontend	Backend
HTML, CSS, JS (DOM)	Node.js, Express
